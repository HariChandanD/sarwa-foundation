# QR Code Image Instructions

## Required Action

You need to save the QR code image provided in the conversation to this location:

```
animal-welfare-ngo/public/qr-code.jpg
```

## Steps to Add the QR Code Image

1. Save the QR code image from the conversation
2. Rename it to `qr-code.jpg`
3. Place it in the `public` folder of the project
4. The image path in the code is: `/qr-code.jpg`

## Image Details

The QR code should be the IDFC FIRST Bank payment QR code with:
- UPI ID: sarwasociety@idfcbank
- Bank: IDFC FIRST Bank
- Supported apps: Paytm, Google Pay, PhonePe, Amazon Pay, CRED, MobiKwik

## Alternative

If you don't have the image yet, you can:
1. Create a placeholder image named `qr-code.jpg` in the `public` folder
2. Replace it with the actual QR code later

The website will work with any image at `/qr-code.jpg`, but for production use, make sure to use the actual payment QR code.
